# Session Log（テンプレート）

このファイルは、あなたのプロジェクトで育てていく「修正ペア」の置き場です。
エージェントの回答方針にあなたから修正が入るたび、下に1行ずつ追記していきます。
正解例より修正例のほうが、エージェントに「境界線」を教えます。ここがこのハーネスで
最も価値のある資産になります（開発時の実例は `validation/harness-development-log.md` を参照）。
